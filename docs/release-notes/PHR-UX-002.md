# PHR-UX-002 Release Note

Replaced the passive application search field with the global Cards command palette. The shell now opens a keyboard-first Universal Asset Picker through Command-K or Control-K, reuses Identity and Eligibility engines, shows canonical artwork, and progresses through printing, finish, and condition before routing to Market Watch or Vendor Workspace.

Search remains identity-only. Market acquisition and workflow mutation stay outside the palette.
