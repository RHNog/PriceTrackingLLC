# PriceTrackingLLC

PriceTrackingLLC is a Professional TCG Decision Operating System for discovering buying opportunities and evaluating card purchases.

The current app includes Hot Opportunities, Watchlists, Vendor Workspace, Vendor Workflow State Machine, Scryfall identity search, market estimates, purchase evaluation, Asset Intelligence, Card Intelligence, Negotiation Ladder, Query Engine, Canonical Resolution, Intent Resolution, Entity Resolution, and Constraint Satisfaction.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Lint

```bash
npm run lint
```
