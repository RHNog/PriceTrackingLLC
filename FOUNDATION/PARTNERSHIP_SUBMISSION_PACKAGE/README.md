# Partnership Submission Package

This folder is intentionally temporary.

Its purpose is to gather every document required to build a professional partnership proposal using another AI system such as Gemini.

Do not treat this folder as the canonical source of truth. The canonical documents remain in `FOUNDATION/` and `docs/`. Future updates should always be made to canonical documents first, then this package should be refreshed by copying updated versions.

## How To Use This Package

Use this package when preparing a professional partnership proposal, deck, executive brief, or related submission for an external strategic partner.

Upload the files to Gemini or another AI system in the recommended order so it receives brand context before business context, and business context before proposal artifacts.

## Recommended Upload Order

1. `Brand_Board_v1.0.png`
2. `Brand_Production_Brief_v1.0.md`
3. `CorporateStyleGuide.md`
4. `PROJECT_PHRONESIS_FOUNDING_CHARTER.md`
5. `BUSINESS_STRATEGY.md`
6. `PRODUCT_ROADMAP.md`
7. `TCGPLAYER_EXECUTIVE_PARTNERSHIP_DECK.md`
8. `TCGPLAYER_ECOSYSTEM_PARTNERSHIP_PROPOSAL.md`
9. `EXECUTIVE_SUMMARY.md`

## Recommended Workflow

1. Upload the brand board and brand production brief first.
2. Upload the corporate style guide.
3. Upload the founding charter.
4. Upload business strategy and product roadmap.
5. Upload the executive partnership deck.
6. Upload the full proposal.
7. Upload the executive summary.
8. Ask the AI to generate or refine the target proposal using the Foundation and brand guidance as governing context.
9. Review all outputs for current capability vs future vision, defensible claims, and partner-safe disclosure.
