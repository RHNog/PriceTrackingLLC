# PHR-UX-005 — Collector Presentation Rules

Collector identity surfaces now show only distinguishing information. Printing Design Facets present as Treatment and Physical Finish presents as Printing.

Standard Treatment and Regular, Normal, Nonfoil, or provider-unavailable Printing are hidden. Borderless Foil, Enchanted, and Cold Foil therefore read naturally without low-information labels. Developer mode retains every canonical value and explains each visibility decision.
