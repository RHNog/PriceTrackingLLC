# PHR-ARCH-009 — Identity Presentation Layer

Added a reusable presentation boundary that lets Project Phronesis reason with canonical ontology concepts while speaking in familiar collector terms.

Command Palette, Vendor results, Watchlist, and capability surfaces now use Printing, Treatment, Finish, Market, and Condition. Developer diagnostics retain the canonical concepts and show every presentation translation.

The Lorcast audit confirmed that Cold Foil is not mapped from rarity or price data. Enchanted remains a Treatment and unavailable Lorcast finish displays Provider Does Not Supply.
