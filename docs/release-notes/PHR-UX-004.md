# PHR-UX-004 Release Note

Added lightweight lifecycle history to every Market Watch membership: added time, initial observed valuation/source, reason, notes, last refresh, last successful refresh, refresh source, watch duration, current observation, and a bounded watch-owned observation series.

Market Watch now shows signed market change since addition and an accessible lightweight sparkline inside expandable row/card details. Target-price difference remains separate. No analytics page, full chart, or pre-watch provider history was added.
